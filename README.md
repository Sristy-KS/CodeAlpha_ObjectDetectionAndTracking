# Object Detection and Tracking using YOLOv8

## Overview

This project performs real-time object detection and tracking using the YOLOv8 model and OpenCV. It captures video from a webcam (or video file), detects multiple objects, assigns tracking IDs, and displays the results in real time.

## Features

- Real-time webcam/video input
- YOLOv8 object detection
- Object tracking with unique IDs
- Bounding boxes with labels
- Real-time display using OpenCV

## Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8
- NumPy

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run

```bash
python object_tracking.py
```

Press **Q** to quit.
